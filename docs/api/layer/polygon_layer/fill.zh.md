## 使用

```javascript
import { PolygonLayer } from '@antv/l7';
const layer = new PolygonLayer();
```

### shape

绘制填充图，shape 为`fill` 常量不支持数据映射

```javascript
layer.shape('fill');
```

### size

填充图无 size 不需要设置 size
